# Renewable Energy Production Dashboard

## 📊 Project Overview
This project is a Power BI dashboard that analyzes renewable energy production across different countries and energy sources such as solar, wind, and hydro.

## 🚀 Features
- Total energy generation analysis
- Country-wise and source-wise comparison
- Interactive dashboard with filters (slicers)
- Clean and structured data using Power Query

## 🛠️ Tools Used
- Power BI
- DAX
- Power Query
- Excel

## 📌 Key Insights
- Identifies top energy-producing countries
- Compares renewable energy sources
- Helps understand energy distribution patterns
